# Wifi_connect_Script

脚本适用于ArchLinux安装。

绕过iwctl 需要的多余操作，可直接封装到Archlinux镜像包内。

### 需要依赖

- networkmanager

- wps_supplicant

  依赖已全部整合进 ```Wifi_connect.sh``` .
